## 프로젝트 개요

StudentManagement는 구글 클래스룸에서 영감을 받아서 제작한 학생 관리 프로그램입니다.

## 프로젝트 상세설명

선생님이 회원가입을 해서 학생을 추가하고 클래스를 추가하여
각 클래스에서 출석 관리를 할 수 있습니다.
출석을 했다가 출석 취소를 할 수 있고, 학생의 출석한 내역을 확인할 수도 있습니다.
